# Sistema de Gerenciamento de Jogos

Projeto desenvolvido em Python para gerenciamento de jogos concluídos, com operações de cadastro, consulta, atualização e remoção de registros.

## Tecnologias

- Python
- JSON

## Funcionalidades

- Cadastro de jogos
- Listagem de jogos
- Atualização de status
- Exclusão de jogos
- Histórico de alterações
- Persistência de dados em JSON

## Como Executar

```bash
python projeto.py
```
